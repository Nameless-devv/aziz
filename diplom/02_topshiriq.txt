TOPSHIRIQ MAZMUNI:

1. BMI mavzusining dolzarbligi va amaliy ahamiyatini asoslash
2. Qishloq xo'jaligida robotlash sohasidagi zamonaviy tadqiqotlarni tahlil qilish
3. Path planning (yo'l rejalashtirisht) algoritmlarini o'rganish va taqqoslash
4. Qishloq xo'jaligi muhiti uchun moslashtirilgan path planning algoritmini ishlab chiqish
5. Algoritmni Python dasturlash tilida amalga oshirish
6. Algoritmning samaradorligini simulyatsiya yordamida tekshirish va natijalarni tahlil qilish

DASTLABKI MA'LUMOTLAR:
- Qishloq xo'jaligi robotlari haqida ilmiy maqolalar va adabiyotlar
- Path planning algoritmlari (A*, Dijkstra, RRT, D*) haqida nazariy asoslar
- Python 3.x dasturlash tili va tegishli kutubxonalar (NumPy, Matplotlib, Pygame)
- Grid-based va graph-based xarita tasviri usullari

GRAFIK MATERIALLAR RO'YXATI:
1. Qishloq xo'jaligi roboti harakatlanish muhitining sxemasi
2. A* algoritmi ishlash jarayonining blok-sxemasi
3. Grid xaritasida yo'l rejalashtirish natijalarining grafik tasviri
4. Algoritmlar taqqoslash natijalari diagrammasi
5. Dastur arxitekturasining UML diagrammasi
